    常用的体言#

    为方便起见，我总结了一些常用的体言，如月份、星期、家人称呼等等。

    年#

    平假名 读音 意义 备注
    半年 はんとし 0 半年
    毎年 まいとし 0 每年 まいねん
    去年 きょねん 1 去年
    今年 ことし 0 今年
    来年 らいねん 0 明年
    数年 すうねん 0 数年
    何年 なんねん 1 哪一年；多少年
