# [動{どう}詞{し}連{れん}用{よう}形{けい}]

连用形是日语动词活用形式中的一种形式，主要用法是后续「ます」等助动词或在句中表示中顿。词尾变化（活用）的方法如下：

- 五段动词的连用形是将词尾的ウ段假名改为イ段假名；

- 一段动词的连用形是将词尾的「る」去掉；

- サ变动词的连用形是将「する」改为「し」；

- カ变动词的连用形是将「[来{く}]る」改为「[来{き}]」。
