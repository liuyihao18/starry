# [動{どう}詞{し}推{すい}量{りょう}形{けい}]

推量形是日语动词活用形中的一种形式，主要用法是后续推量助动词「う」或「よう」，(1);表示说话人要进行该动作的决心，意志；(2)
;表示向对方进行号召、劝诱。词尾变化（活用）的方法如下：

- 五段动词的推量形是将词尾由ウ段假名改为オ段假名；

- 一段动词去掉词尾的「る」即可；

- サ变动词是将「する」改为「し」；

- カ变动词是将「[来{く}]る」改为「[来{こ}]」。

> 事实上，“推量形”在语法上被归类为“未然形”。
