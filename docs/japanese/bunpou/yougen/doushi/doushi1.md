# [動{どう}詞{し}未{み}然{ぜん}形{けい}]

未然形是日语动词活用形中的一种形式，主要用法是后续助动词「ない（ぬ）」、「せる（させる）」和「れる（られる）」。词尾变化（活用）的方法如下：

- 五段动词的未然形是将ウ段词尾改为ア段；

- 一段动词的未然形是将词尾的「る」去掉；

- サ变动词的未然形是将「する」变成「し」；

- カ变动词的未然形是将「[来{く}]る」改为「[来{こ}]」。
