# 常用的体言

| 毎年 | まいとし | (0); |      每年      | まいねん |
| 去年 | きょねん | (1); |      去年      |          |
| 今年 |  ことし  | (0); |      今年      |          |
| 来年 | らいねん | (0); |      明年      |          |
| 数年 | すうねん | (0); |      数年      |          |
| 何年 | なんねん | (1); | 哪一年；多少年 |          |
