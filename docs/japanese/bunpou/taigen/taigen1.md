# [代{だい}名{めい}詞{し}]

日语中也有各种人称代词和指示代词，但是用法与中文不完全一样，需要注意体会。

## 一、人称代词

人称代词分为自称、对称和他称三种，性别不同，用词也不同。

## 二、指示代词

日语的指示代词分为「こそあど」系词汇，对应近称、中称、远称和不定称。
